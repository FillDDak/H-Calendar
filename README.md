# H-Calendar

## 변경점
```
MySQL 스키마, 유저, 비밀번호 통일
스키마 Charset utf8mb4로 설정
```

### 프론트엔드 폴더와 백엔드 폴더에서 터미널로 입력
```
npm install
```

### 프론트엔드 실행
```
npm run serve
```

### 백엔드 실행
```
npm run start
```
